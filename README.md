# Vite + React + TypeScript

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Libraries included
- Emotion
- Material UI

## Install dependencies

#### `npm install`

## Available Scripts

#### `npm run dev` - starts a local web server with HMR for development
#### `npm run preview` - starts a local web server that serves the built solution from ./dist for previewing
#### `npm run build` - builds the project and outputs to the folder ./dist
#### `npm run lint` - runs ESLint for the whole project
#### `npm run lint:fix` - runs ESLint for the whole project and fix possible issues
